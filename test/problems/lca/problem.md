# 根付き木の LCA

**Tags:** `graph` `tree_lca`

**Time Limit:** 2.0 sec

## 問題文
$N$ 頂点の根付き木が与えられます。頂点 $0$ が根です。
$Q$ 個のクエリが与えられるので、各クエリについて頂点 $u$ と $v$ の最小共通祖先 (LCA) を求めてください。

## 制約
- $1 \le N \le 500,000$
- $1 \le Q \le 500,000$
- $0 \le p_i < i$
- $0 \le u_i, v_i < N$

## 入力
```text
$N \quad Q$
$p_1 \quad p_2 \quad \dots \quad p_{N-1}$
$u_0 \quad v_0$
$u_1 \quad v_1$
$\vdots$
$u_{Q-1} \quad v_{Q-1}$
```

## 出力
```text
各クエリに対して、最小共通祖先の頂点番号を改行区切りで出力してください。
```

## サンプル 1
### 入力例
```text
5 3
0
0
1
1
0 4
4 3
2 2
```
### 出力例
```text
0
1
2
```
