# 定数

競技プログラミングで頻出する数値定数や文字定数を提供します。

## 定数一覧
- `mod998`: $998244353$
- `mod107`: $1000000007$
- `mod109`: $1000000009$
- `mod31`: $2147483647$
- `mod61`: $(1 \ll 61) - 1$
- `iINF`: $1001001001$
- `liINF`: $(1 \ll 60) + 1$
- `EL`: `
` (改行)
- `SPA`: ` ` (空白)
