## 概要

非負整数の集合を管理するデータ構造です。
64-ary tree (64分木) を用いて、要素の追加・削除・検索・前後の要素探索をビット演算で高速に行います。

## コンストラクタ

```cpp
FastSet(i32 n)
```

- $[0, n)$ の範囲を管理する空の集合を構築します。
- **制約**: $n \ge 0$
- **計算量**: $O(n / 64)$

## メソッド

### insert

```cpp
void insert(i32 x)
```

- 集合に要素 $x$ を追加します。
- **制約**: $0 \le x < n$
- **計算量**: $O(\log_{64} n)$

### erase

```cpp
void erase(i32 x)
```

- 集合から要素 $x$ を削除します。
- **制約**: $0 \le x < n$
- **計算量**: $O(\log_{64} n)$

### contains

```cpp
bool contains(i32 x) const
```

- 集合に要素 $x$ が含まれるか判定します。
- **制約**: なし（$0 \le x < n$ の範囲外の場合は `false` を返します）
- **計算量**: $O(1)$

### next

```cpp
i32 next(i32 x) const
```

- $x$ 以上の最小の要素を返します。そのような要素が存在しない場合は $n$ を返します。
- **制約**: なし
- **計算量**: $O(\log_{64} n)$

### prev

```cpp
i32 prev(i32 x) const
```

- $x$ 以下の最大の要素を返します。そのような要素が存在しない場合は $-1$ を返します。
- **制約**: なし
- **計算量**: $O(\log_{64} n)$
